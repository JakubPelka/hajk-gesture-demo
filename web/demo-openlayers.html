Gesture control help

Use hand gestures in front of the camera to control the map. The camera preview is mainly for checking hand position in the camera field of view.

Current gestures

  • Thumb Up — activate gesture control
  • Thumb Down — deactivate gesture control
  • Open Palm — pan the map
  • Pinch close — zoom in
  • Pinch spread — zoom out
  • Pointing Up — laser pointer
  • Pointing Up + air tap — click ripple
  • ILoveYou — reset map
  • Phone gesture — open / close this help

Browser controls

  • Show / Hide camera — toggle camera preview panel
  • Switch camera — try next available camera
  • Reset map — reset OpenLayers view
  • Quit Python — stop local Python app

Keyboard shortcuts A/R/Q/ESC are intentionally disabled globally, so future map text input remains safe.
